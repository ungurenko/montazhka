# 🎬 Монтажка

Простая и лёгкая монтажка для Mac. Открывается мгновенно, всё по-русски, ничего лишнего:
добавил клипы → увидел звуковую волну → вырезал паузы (сам или автоматикой) → сохранил MP4.

## ✨ Что умеет

- **Клипы на одной ленте** — как бусины на нитке: режь, удаляй, меняй местами
- **Звуковая волна на каждом клипе** — паузы видно глазами
- **Автопоиск пауз** — программа сама находит тишину и подсвечивает её; ты проверяешь галочками и вырезаешь одной кнопкой
- **Настройки поиска**: чувствительность (дБ), минимальная длина паузы, «воздух» по краям
- **Экспорт в MP4** — три уровня качества, прогресс, «Показать в Finder»
- **Всё сохраняется само** — как в Заметках; исходные видео никогда не изменяются
- **Отмена** — Cmd+Z / Shift+Cmd+Z на каждое действие

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---|---|
| Пробел | Плей / пауза |
| S | Разрезать по курсору |
| Delete | Удалить выбранный клип |
| ← / → | Кадр назад / вперёд (с Shift — секунда) |
| Cmd+Z | Отменить |

## 🔧 Для разработчика (и AI-агентов)

Нативное приложение: Swift 6 + SwiftUI + AVFoundation, macOS 14+. Собирается **без Xcode** —
достаточно Command Line Tools.

```bash
scripts/build-app.sh            # собрать build.noindex/Монтажка.app
scripts/build-app.sh --install  # собрать и поставить в /Applications
swift build && .build/debug/Montazhka --selftest   # самопроверка движка
swift scripts/make-icon.swift   # перегенерировать иконку
```

### Архитектура

```
Sources/Montazhka/
├── main.swift              # вход: GUI или --selftest
├── App.swift               # SwiftUI-приложение, навигация (AppModel)
├── Models/                 # Project, Clip, настройки поиска пауз; JSON-хранилище
├── Engine/
│   ├── EditorController    # сердце: плеер, правки, отмена, автосохранение
│   ├── CompositionBuilder  # склейка клипов в AVMutableComposition
│   ├── TimelineOps         # чистая математика резки (тестируется)
│   ├── WaveformStore       # RMS-громкость окнами 10 мс + кэш на диске
│   ├── SilenceDetector     # поиск тишины по порогу/длине/отступам
│   └── Exporter            # AVAssetExportSession → MP4
├── Views/                  # StartView, EditorView, TimelineView, PausePanel, ExportSheet
└── SelfTest/               # самопроверка движка на сгенерированном видео
```

Ключевые принципы:
- Исходники не трогаем: клип = ссылка на файл + границы (start/end)
- Любая правка = снимок в стек отмены → пересборка композиции → автосохранение (0.5 сек дебаунс)
- Проекты: `~/Library/Application Support/Montazhka/Projects/*.json`
- Кэш волн: `~/Library/Application Support/Montazhka/Waveforms/*.f32`
